# Hospital Appointment Scheduler
A web app to manage doctor appointments. Admins can add doctors and patients can book appointments based on specialization.
## Live Link
https://hospital-appointment-six-xi.vercel.app
## Features:
- Admins can add doctors with a specialization and a daily patient limit
- Patients can book an appointment by entering a specialization
- The app automatically assigns the doctor with the least appointments
- Booking is rejected if all doctors of that specialization are full
## Tech used
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL on Neon
- Authentication: JWT
